# Remove-LWAzureVM

## SYNOPSIS
Remove an Azure VM

## SYNTAX

```
Remove-LWAzureVM [-Name] <String> [-AsJob] [-PassThru] [<CommonParameters>]
```

## DESCRIPTION
Remove an Azure VM

## EXAMPLES

### Example 1
```powershell
PS C:\> Remove-LWAzureVm -ComputerName DC01 -AsJob
```

Removes the host DC01 in a background job
