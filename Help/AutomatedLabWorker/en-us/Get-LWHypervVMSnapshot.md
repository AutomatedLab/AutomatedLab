# Get-LWHypervVMSnapshot

## SYNOPSIS
Find snapshots of Hyper-V VMs

## SYNTAX

```
Get-LWHypervVMSnapshot [[-VMName] <String[]>] [[-Name] <String>] [<CommonParameters>]
```

## DESCRIPTION
Find snapshots of Hyper-V VMs

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-LWHypervVMSnapshot -VMName DSCDC01
```

List all existing snapshots of DSCDC01
