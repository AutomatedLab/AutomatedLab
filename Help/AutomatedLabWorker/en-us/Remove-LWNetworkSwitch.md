# Remove-LWNetworkSwitch

## SYNOPSIS
Remove a Hyper-V network switch

## SYNTAX

```
Remove-LWNetworkSwitch [-Name] <String> [<CommonParameters>]
```

## DESCRIPTION
Remove a Hyper-V network switch

## EXAMPLES

### Example 1
```powershell
PS C:\> Remove-LWNetworkSwitch -Name LabNetwork
```

Remove the virtual switch LabNetwork
