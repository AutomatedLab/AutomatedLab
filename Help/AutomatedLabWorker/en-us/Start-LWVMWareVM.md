# Start-LWVMWareVM

## SYNOPSIS
Start a VMWare VM

## SYNTAX

```
Start-LWVMWareVM [-ComputerName] <String[]> [[-DelayBetweenComputers] <Int32>] [<CommonParameters>]
```

## DESCRIPTION
Start a VMWare VM

## EXAMPLES

### Example 1
```powershell
PS C:\> Start-LWVMWareVM Host1,Host2
```

Start Host1 and Host2 in parallel
