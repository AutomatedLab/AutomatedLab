# Remove-LWHypervVM

## SYNOPSIS
Remove a Hyper-V VM

## SYNTAX

```
Remove-LWHypervVM [-Name] <String> [<CommonParameters>]
```

## DESCRIPTION
Remove a Hyper-V VM

## EXAMPLES

### Example 1
```powershell
PS C:\> Remove-LWHypervVM -Name DC01
```

Deletes the VM DC01

## PARAMETERS

### -Name
