# Stop-LWVMWareVM

## SYNOPSIS
Stop a VMWare VM

## SYNTAX

```
Stop-LWVMWareVM [-ComputerName] <String[]> [<CommonParameters>]
```

## DESCRIPTION
Stop a VMWare VM

## EXAMPLES

### Example 1
```powershell
PS C:\> Stop-LWVMWareVM -ComputerName (Get-LabVM)
```

Stops all VMWare VMs in the lab

## PARAMETERS
