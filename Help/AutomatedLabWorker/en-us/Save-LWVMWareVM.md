# Save-LWVMWareVM

## SYNOPSIS
Save the state of a VMWare VM

## SYNTAX

```
Save-LWVMWareVM [-ComputerName] <String[]> [<CommonParameters>]
```

## DESCRIPTION
Save the state of a VMWare VM

## EXAMPLES

### Example 1
```powershell
PS C:\> Save-LWVMWareVM -ComputerName (Get-LabVm)
```

Saves all lab VMs

## PARAMETERS

### -ComputerName
