# Enable-LWHypervVMRemoting

## SYNOPSIS
Enable CredSSP on a Hyper-V VM

## SYNTAX

```
Enable-LWHypervVMRemoting [-ComputerName] <String[]> [<CommonParameters>]
```

## DESCRIPTION
Enable CredSSP on a Hyper-V VM

## EXAMPLES

### Example 1
```powershell
PS C:\> Enable-LWHypervVMRemoting (Get-LabVM)
```

Enable CredSSP on all lab VMs

## PARAMETERS

### -ComputerName
