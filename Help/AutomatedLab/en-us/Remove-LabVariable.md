# Remove-LabVariable

## SYNOPSIS
Remove AutomatedLab variables

## SYNTAX

```
Remove-LabVariable [<CommonParameters>]
```

## DESCRIPTION
Removes all variables matching  'AL_(\[a-zA-Z0-9\]{8})+\[-.\]+(\[a-zA-Z0-9\]{4})+\[-.\]+(\[a-zA-Z0-9\]{4})+\[-.\]+(\[a-zA-Z0-9\]{4})+\[-.\]+(\[a-zA-Z0-9\]{12})'

## EXAMPLES

### Example 1
```powershell
PS C:\> Remove-LabVariable
```
