# Remove-DeploymentFiles

## SYNOPSIS
Remove deployment data

## SYNTAX

```
Remove-DeploymentFiles
```

## DESCRIPTION
Removes the following files and folders used to deploy the machines from all running lab machines:
C:\unattend.xml
C:\WSManRegKey.reg
C:\DeployDebug
C:\AdditionalDisksOnline.ps1

## EXAMPLES

### Example 1
```powershell
PS C:\> Remove-DeploymentFiles
```

Removes the following files and folders used to deploy the machines from all running lab machines:
C:\unattend.xml
C:\WSManRegKey.reg
C:\DeployDebug
C:\AdditionalDisksOnline.ps1
