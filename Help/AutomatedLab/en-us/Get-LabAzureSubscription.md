# Get-LabAzureSubscription

## SYNOPSIS
Get an Azure subscription

## SYNTAX

```
Get-LabAzureSubscription [<CommonParameters>]
```

## DESCRIPTION
Gets all subscriptions for a given lab

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-LabAzureSubscription
```

Gets all subscriptions for a given lab
