# Install-LabADDSTrust

## SYNOPSIS
Create forest trust

## SYNTAX

```
Install-LabADDSTrust [<CommonParameters>]
```

## DESCRIPTION
Creates DNS forwarders and a bidirectional trust for all forests in the current lab

## EXAMPLES

### Example 1
```powershell
PS C:\> Install-LabADDSTrust
```

Creates DNS forwarders and a bidirectional trust for all forests in the current lab
