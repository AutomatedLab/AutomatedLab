# Get-LabVMSnapshot

## SYNOPSIS
Get the existing checkpoints of a lab VM

## SYNTAX

```
Get-LabVMSnapshot [[-ComputerName] <String[]>] [[-SnapshotName] <String>] [<CommonParameters>]
```

## DESCRIPTION
Get the existing checkpoints of a lab VM

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-LabVMSnapshot -ComputerName DC1
```

List all checkpoints of DC1
