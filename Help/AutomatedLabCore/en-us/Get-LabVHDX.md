# Get-LabVHDX

## SYNOPSIS
Get lab disks

## SYNTAX

### ByName
```
Get-LabVHDX -Name <String[]> [<CommonParameters>]
```

### All
```
Get-LabVHDX [-All] [<CommonParameters>]
```

## DESCRIPTION
Gets lab disk files either by name or returns all lab disks

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-LabVhdx
```

Return all lab VHDX files
