# Get-LabVMUptime

## SYNOPSIS
Get uptime

## SYNTAX

```
Get-LabVMUptime [-ComputerName] <String[]> [<CommonParameters>]
```

## DESCRIPTION
Gets the VMs uptime by checking the last boot time remotely via WMI

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-LabVMUptime -ComputerName FS1
```

Get the uptime of FS1
