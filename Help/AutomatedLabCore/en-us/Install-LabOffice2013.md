# Install-LabOffice2013

## SYNOPSIS
Install Microsoft Office 2013

## SYNTAX

```
Install-LabOffice2013 [<CommonParameters>]
```

## DESCRIPTION
Installs Microsoft Office 2013 on designated lab machines

## EXAMPLES

### Example 1
```powershell
PS C:\> Install-LabOffice2013
```

Installs Microsoft Office 2013 on designated lab machines
