# Reset-AutomatedLab

## SYNOPSIS
Reset the lab

## SYNTAX

```
Reset-AutomatedLab [<CommonParameters>]
```

## DESCRIPTION
Resets the lab by removing the current lab and unloading all modules

## EXAMPLES

### Example 1
```powershell
PS C:\> Reset-AutomatedLab
```

Resets the lab by removing the current lab and unloading all modules
