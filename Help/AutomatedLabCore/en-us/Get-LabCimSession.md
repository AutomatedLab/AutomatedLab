# Get-LabCimSession

## SYNOPSIS
Cmdlet to list all or specific CIM sessions

## SYNTAX

```
Get-LabCimSession [[-ComputerName] <String[]>] [-DoNotUseCredSsp] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet to list all or specific CIM sessions

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-LabCimSession
```

List all currently open CIM sessions in your lab
