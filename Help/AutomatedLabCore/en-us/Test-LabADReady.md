# Test-LabADReady

## SYNOPSIS
Test if lab ADWS are ready for scripting

## SYNTAX

```
Test-LabADReady [-ComputerName] <String> [<CommonParameters>]
```

## DESCRIPTION
Test if lab ADWS are ready for scripting

## EXAMPLES

### Example 1
```powershell
PS C:\> Test-LabADReady -ComputerName DC1
```

Test if lab ADWS are ready on DC1 for scripting
