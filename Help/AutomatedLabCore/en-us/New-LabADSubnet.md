# New-LabADSubnet

## SYNOPSIS
Create new AD subnet

## SYNTAX

```
New-LabADSubnet [-PassThru] [<CommonParameters>]
```

## DESCRIPTION
Creates a new Active Directory subnet for the site Default-First-Site-Name

## EXAMPLES

### Example 1
```powershell
PS C:\> New-LabADSubnet 
```

Creates a new Active Directory subnet for the site Default-First-Site-Name
