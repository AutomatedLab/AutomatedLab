# Get-Lab

## SYNOPSIS
Show lab data

## SYNTAX

```
Get-Lab [-List] [<CommonParameters>]
```

## DESCRIPTION
Return the lab data for the current lab or list all existing labs

## EXAMPLES

### Example 1
```powershell
Import-Lab MyLabName
$LabData = Get-Lab
```

Imports a lab and stores the lab data in the variable LabData

## PARAMETERS
