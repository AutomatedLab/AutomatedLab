# Get-UnattendedContent

## SYNOPSIS
Return the current unattend content that has been created

## SYNTAX

```
Get-UnattendedContent [<CommonParameters>]
```

## DESCRIPTION
Return the current unattend content that has been created

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-UnattendedContent
```

Returns whatever Unattend content has been created.
