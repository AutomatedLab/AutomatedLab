# Save-LabRecipe

## SYNOPSIS
Save a lab recipe to disk

## SYNTAX

```
Save-LabRecipe [-Recipe] <PSObject> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Save a lab recipe to disk

## EXAMPLES

### Example 1
```powershell
LabRecipe SuperEasy {
    DeployRole = 'Domain', 'PKI'
} | Save-LabRecipe
```

Saves the recipe SuperEasy
